## 📖 About

CP Coach is an AI-powered Competitive Programming platform designed to eliminate random practice and provide a structured learning experience.

Instead of manually searching for problems, CP Coach analyzes your competitive programming profiles, identifies your strengths and weaknesses, and generates personalized daily practice sheets powered by AI.

Whether you're preparing for Codeforces contests, improving your CodeChef rating, or solving LeetCode problems, CP Coach acts as your personal mentor.

- Current Rating
- Maximum Rating
- Contest History

- 2 Codeforces Problems
- 2 CodeChef Problems
- 1 LeetCode Problem

No random recommendations.

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Tauhidrana/cp-coach.git
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build production

```bash
npm run build
```
